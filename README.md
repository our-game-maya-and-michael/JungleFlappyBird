# Jungle Flappy Bird
In this repository I remade the game of FlappyBird with my little spin to it:
Try to navegate in the jungle by being a big boy of a bird and see the dangers of flaying to close to tree from first hand experiance and try to escape the endless jungle ! [to play the game on itch.io](https://mayamichael.itch.io/jungleflappybird)
